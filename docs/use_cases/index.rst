================
Use Cases
================
Elemeta is currently focused on NLP use cases and can help you explore, monitor, and extract features from unstructured data through enriched tabular representations. As the package expands, additional use cases for vision, embeddings, audio, etc., will be added, as well as OOTB integrations between Elemeta and MLOps tools.
